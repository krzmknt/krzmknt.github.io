Basic commands

Session

  • New: tmux new [-s <session-name>] [-n <window-name>]
  • Attach: tmux a [-t <session-name>]
  • Detach: Prefix+d
  • Kill: tmux kill-ses [-t <session-name>]
  • Kill the all sessions: tmux kill-server
  • Rename: tmux rename -t <old-name> <new-name> or Prefix+$
  • List
    • tmux ls
    • Prefix+s

Window(Prefix+)

  • New/Kill
    • c : Create window
    • & : Close current window
  • Rename
    • , : Rename current window
  • Switch
    • p : Previous window
    • n : Next window
    • b [0..9] : Select window by number

Pain(Prefix+)

  • New/Kill
    • % : Split pane horizontally
    • " : Split pane vertically
    • x : Close current pane
  • Alter
    • { : Move the current pane left
    • } : Move the current pane right
    • z : Toggle pane zome
    • space : Toggle between pane layouts
  • Convert
    • ! : Conver pane into a window
  • Switch
    • o : Switch to next pane
    • q [0..9] : Select pane by number
    • ; : Toggle last active pane
    • arrow : Switch to pane to the direction
