Basic commands

Session

  • tmux new [-s <session-name>] [-n <window-name>]: Create a new settion
  • tmux a [-t <session-name>]: Attach the exsiting session
  • tmux a -d [-t <session-name>]: Detach the existing session and then attach it
  • Prefix+d: Detach the current session
  • tmux kill-ses [-t <session-name>]: Kill the session
  • tmux kill-server: Kill the all session
  • tmux rename -t <old-name> <new-name> or Prefix+$: Rename the session
  • tmux ls or Prefix+s: List the existing sesions

Window(Prefix+)

  • c: Create a window
  • &: Close the current window
  • ,: Rename current window
  • p: Move to the previous window
  • n: Move to the next window
  • b [0..9]: Select window by number

Pain(Prefix+)

  • %: Split the pane horizontally
  • ": Split the pane vertically
  • x: Close the current pane
  • {: Move the current pane left
  • }: Move the current pane right
  • z: Toggle the pane zome
  • [space]: Toggle between pane layouts
  • !: Conver the pane into a window
  • o: Switch to the next pane
  • q [0..9]: Select the pane by number
  • ;: Toggle the last active pane
  • [arrow]: Switch to the pane to the direction
