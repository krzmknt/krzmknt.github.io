Install

$ sudo apt install tmux

$ brew install tmux

Architecture

  • tmux
    • Session
      • Window
        • Pain

Commands

Session

  • Make
    • tmux new|new-session [-s ]
    • :new
      • Start a new session.
      • If you don’t give any option, subcommand ‘new|new-session’ can be omitted.
  • Attach
    • tmux a [-t ]
      • Attach to last session
      • With the option -t, attach to a session with the name .
  • Kill
    • tmux kill-ses [-t ]
      • Kill the lru session.
      • With option -t, you can specify a session.
    • tmux kill-server
      • Kill the all sessions.
  • Rename
    • C-b+$, :rename-session : Chage the session name
    • tmux rename -t : Change the session name
  • List
    • tmux ls
    • Ctrl+b s

Window

  • tmux new -s mysession -n mywindow
    • Start a new session with the name mysession and window mywindow.
  • Ctrl+b
    • Make/Kill
      • c : Create window
      • & : Close current window
    • Rename
      • , : Rename current window
    • Switch
      • p : Previous window
      • n : Next window
      • b [0..9] : Select window by number

Pain

  • Ctrl+b
    • Make/Kill
      • % : Split pane horizontally
      • " : Split pane vertically
      • x : Close current pane
    • Alter
      • { : Move the current pane left
      • } : Move the current pane right
      • z : Toggle pane zome
      • space : Toggle between pane layouts
    • Convert
      • ! : Conver pane into a window
    • Switch
      • o : Switch to next pane
      • q [0..9] : Select pane by number
      • ; : Toggle last active pane
      • arrow : Switch to pane to the direction
