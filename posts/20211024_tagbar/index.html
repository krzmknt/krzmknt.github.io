【vim】Jump with ctags

Install ctags

brew install --HEAD universal-ctags/universal-ctags/universal-ctags

Create a tag file

Execute ctags -Rf .tags to generate .tags. ctags analyzes the source codes on the current directly recursively.

Try to jump

Execute :set tags=.tags on vim. (It is recomended to register this command to .vimrc) Execute a vim command :ta <word> or type ctrl+] on a <word> to jump to the definition.
